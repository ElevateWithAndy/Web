---
title: "How to use Markdown blog posts"
date: "2026-05-31"
description: "Use Markdown files for individual blog posts in Astro."
---

Each blog post lives in `src/pages/blog/` as its own Markdown file. Astro will render these automatically as pages, and the blog index lists them from the same folder.

Use frontmatter to control the title, date, and description for each post.
