# Web

This repository contains an Astro site for the Elevate with Andy GitHub Pages project.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

A GitHub Actions workflow deploys the generated `dist/` output to the `gh-pages` branch.

GitHub Pages URL:

`https://elevatewithandy.github.io/Web`
